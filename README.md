# AI Recruiter

## Local URLs

- Frontend: http://localhost:5173
- Backend API: http://localhost:5000
- Backend docs: http://localhost:5000/docs

This project uses port `5000` for the FastAPI backend.

## Run Backend

```bash
cd backend
uvicorn app.main:app --host 127.0.0.1 --port 5000 --reload
```

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```
